# 數位錯題本

## Proposal Report

### 動機與目標
我的微積分一直以來都不太好，考試和作業也常常錯誤百出，所以我希望能夠有更有系統、更方便查看自己曾經錯過的題目或考試壹定會考的經典題型的方法，不然每次都要一一去翻我之前做的筆記很麻煩也消磨我的動機。

### 預期功能
這個程式碼要有使用者介面(tkinter，也就是單機給我自己用)，且至少包含五個功能：
1.歸檔，讓我能夠依照單元分配資料夾來存放題目


### 使用技術
使用文字編輯器sublime，且引進python的函式庫tkinter來完成使用者介面。
關於app關掉要如何儲存的問題，我會想辦法用上學期學的file input功能來完成。
我感覺自己一定會在某些環節卡住，所以會請Gemini來幫忙

### 時程規劃

5/13前完成除了關掉app後儲存的大部分功能
5/26前想辦法搞定儲存的功能並完成專案
